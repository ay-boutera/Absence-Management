# Absence Management

A full-stack absence management system composed of three separate repositories, brought together here as an organisation hub.

## Repositories

| Repository | Description |
|---|---|
| [Absence-Management-frontend](https://github.com/ay-boutera/Absence-Management-frontend) | Web front-end (user interface) |
| [Absence-Management-backend](https://github.com/ay-boutera/Absence-Management-backend) | REST API / server-side logic |
| [Absence-Management-mobile](https://github.com/ay-boutera/Absence-Management-mobile) | Mobile application |

## Getting Started

This repository uses [Git submodules](https://git-scm.com/book/en/v2/Git-Tools-Submodules) to reference each component. Clone everything at once with:

```bash
git clone --recurse-submodules https://github.com/ay-boutera/Absence-Management.git
```

If you have already cloned the repo without submodules, initialise them with:

```bash
git submodule update --init --recursive
```
